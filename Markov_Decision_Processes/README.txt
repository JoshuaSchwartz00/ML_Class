Libraries used: matplotlib, numpy, mdptoolbox-hiive

1. Clone the github repository with: "git clone https://github.com/JoshuaSchwartz00/ML_Class.git"
2. Clone the hiive-mdptoolbox github repository into your python site-packages directory with: "git clone https://github.com/hiive/hiivemdptoolbox.git"
3. Use pip to install all the libraries listed with: "pip install <libraries>"
4. Access the ML_Class/Markov_Decision_Processes directory
5. Run the code within by inputting: "py -3 markov.py"

The graphs will automatically saved as pngs in the Markov_Decision_Processes directory.

Analysis is included in the report.
